2.) Get Cases for the Indian States



http://C0V1DTracker/covid /data/getIndianStateData

i. This link provides the JSON data for the cases of all the states that are registered in the State Indian Constituency and are providing constant cases data to the Indian Govt.
ii. This Link provided Data for the indivisual States of the country and provides data of total active,confirmed,recovered,deaths and an array of data on the districts of that state which include district confirmed,recovered and death cases
iii. The JSON Object for the API call is represented by the given object class :
1.
[IndianStates : {

confirmed : Integer,
deaths : Integer,

3.) Get Cases for Specific District of any Indian State



http://C0V1DTracker/covid /data/user/{state}/{city}

i. This link provides the JSON data for the cases for any district that is registered in the State Indian Constituency and are providing constant cases data to the Indian Govt.
ii. This Link Provides Data for the specified District of the country.
iii. The JSON Object for the API call is represented by the given object class :
1.
UserModelStatsDto : {
city : String,
state : String,
stateConfirmedCases : Integer,
stateActiveCases : Integer,
stateRecoveredCases : Integer,
stateDeathCases : Integer,
districtConfirmedCases : Integer,
todaysCaseCount : Integer
};

iv. The following API call will generate the desired List expected by the user :
